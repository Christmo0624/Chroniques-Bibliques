# Chroniques Bibliques — Joseph, Gouverneur d'Égypte

Un jeu d'aventure narratif interactif basé sur le récit biblique de Joseph en Égypte (Genèse 37-50). Prenez des décisions morales difficiles, résolvez des énigmes logiques et textuelles en vous appuyant sur le Codex biblique, et observez l'évolution des vertus de Joseph (Foi, Sagesse, Courage, Discernement, Compassion) en temps réel.

Développé pour la plateforme **Djata X**.

---

## 🌟 Fonctionnalités

- **Moteur de Dialogue Immersif** : Récit fluide avec effet d'écriture progressive (typewriter effect) et surbrillance des visages animés (SVG).
- **Codex Biblique Intégré** : Base de données de versets clés (Genèse & Psaumes) avec barre de recherche en temps réel et surbrillance pour vous aider à résoudre les énigmes.
- **Puzzles Variés** : 
  - Reconstitution de versets (Psaume 27:10).
  - Identification de marchandises historiques (Galaad).
  - Textes à trous bibliques.
  - Correspondances d'interprétation de songes (Échanson, Panetier, Pharaon).
- **Graphique Radar Réactif (SVG)** : Modélisation en temps réel du caractère de Joseph selon vos choix moraux.
- **Synthétiseur Sonore natif** : Effets audio rétro générés en temps réel par l'API Web Audio (sans aucun fichier externe à charger).
- **Sauvegarde Automatique** : Persistence de la progression locale (`localStorage`).

---

## 🛠️ Technologies Utilisées

- **Structure** : HTML5 (balises sémantiques).
- **Style** : CSS3 natif (Design responsive, palette parchemin/navy/or, micro-animations, verre dépoli).
- **Logique** : JavaScript ES6+ moderne (State management local).
- **Audio** : Web Audio API pour la synthèse sonore en temps réel.
- **Graphiques** : Dessin vectoriel et animations SVG.

---

## 🚀 Comment y jouer ?

1. Clonez ce dépôt.
2. Ouvrez simplement le fichier `chroniques_bibliques_mockup.html` dans n'importe quel navigateur moderne.
3. Aucun serveur ou installation de base de données externe n'est requis !
